# Cryprice

Монорепозиторий: публичный API крипто-цен, фронтенд, лендинг, примеры интеграций и документация.

## Структура

| Путь | Назначение |
|------|------------|
| `apps/web` | Публичная/демо-версия frontend |
| `apps/marketing` | Лендинг и страницы продукта |
| `packages/api-client` | Публичный SDK / HTTP-клиент к API |
| `packages/shared` | Общие типы, утилиты, константы |
| `examples/*` | Минимальные примеры использования |
| `docs` | Архитектура, API, роадмап, локальный сетап |
| `backend-public` | Публичный backend (источник правды для API) |

Подробнее: [docs/setup.md](docs/setup.md).
