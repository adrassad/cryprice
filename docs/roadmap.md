# Roadmap

1. **MVP** — health, один агрегатор котировок, кэш, минимальный `api-client`.
2. **Публичная демо** — `apps/web` с живыми данными.
3. **Маркетинг** — `apps/marketing`, SEO, страницы документации API.
4. **Надёжность** — rate limits, метрики, SLA для публичного API.
5. **Расширение** — исторические свечи, WebSocket (при необходимости).

Даты и приоритеты уточняются отдельно.
